# Phantasma Pathtracer

My repository for my Phantasma Pathtracer built along with the Advanced Graphics Masterclass at BUas, given by Jacco Bikker.

### Controls
- WASD - Move camera
- Space/LCtrl - Move up/down
- F1 - Toggle cursor

### Current features
- GPU Acceleration via OpenCL

Planned:
- Model loading
