### Controls
- WASD - Move camera
- Space/LCtrl - Move up/down
- F1 - Toggle cursor / UI
- P - Screenshot

### Current features
- GPU Acceleration via OpenCL
- Live shader recompilation
- SAH BVH
- EXR/HDRI Support
- Wavefront Pathtracing (Currently WIP)
